Onion Browser is a minimal web browser that encrypts and tunnels web traffic through the Tor anonymity network. It is only intended to provide a light web browser "shell" for privacy-seeking users and is not intended to be a full-featured web browser. For more information about Onion Browser and the benefits of using this app, visit the official Onion Browser site.

This is open source software: source code for this application can be found at https://github.com/mtigas/iOS-OnionBrowser — see below for license information.
